## User file permissions

Summary of permission using octal numbers.

| Symbolic | Octal  | English              |
| :------: | :----: | :------------------: |
|   - - -  |   0    | No Permission        |
|   - - x  |   1    | Execute              |
|   - - w  |   2    | Write                |
|   - w x  |   3    | Write & Execute      |
|   r - -  |   4    | Read                 |
|   r - x  |   5    | Read & Execute       |
|   - r w  |   6    | Read & Write         |
|   r w x  |   7    | Read, Write, Execute |
